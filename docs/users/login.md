# 通常のログイン手順

このシステムの通常ログインは、`ドメイン + ユーザーID + パスワード` による認証のあと、メールで受け取る OTP を入力する 2 段階方式です。

![通常ログイン画面プレースホルダー](assets/user-login-flow.svg)

## ログイン前に準備するもの

- 会社環境のドメイン名
- ユーザーID
- パスワード
- OTP を受信できるメールアドレス

## 手順

1. ログイン画面を開きます。
2. `Domain` に会社環境のドメイン名を入力します。
3. `Username` にユーザーIDを入力します。
4. `Password` にパスワードを入力します。
5. ログインボタンを押します。
6. 登録メールアドレス宛に OTP が送信されます。
7. OTP 入力画面で、受信したコードを入力します。
8. 認証が成功すると、対象会社のホーム画面に移動します。

## 補足

- ログイン画面ではメールアドレスではなく、ユーザーIDで認証します。
- OTP はメール送信されます。
- OTP 認証後、会社ごとのサブドメイン画面へ遷移します。

## ログインできないときの確認ポイント

- ドメイン名が正しいか
- ユーザーIDとパスワードが正しいか
- OTP メールが迷惑メールに入っていないか
- メールアドレスが最新か

## 画面イメージに対応する入力項目

- Domain: 利用会社の環境識別子
- Username: ログイン用ユーザーID
- Password: ログインパスワード
- OTP: メールで受信した確認コード
