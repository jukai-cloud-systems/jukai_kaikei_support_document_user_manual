# 樹海クラウド会計支援 ユーザーマニュアル

本誌は、`樹海クラウド会計支援` のユーザー向け手順書です。  

## システムについて

- 請求書や領収書などの書類ファイルをアップロードする
- OCR の読み取り結果をもとに、伝票の基本情報を登録する
- 申請情報と仕訳情報を整えて、支払申請を提出する
- 支払申請を承認者が確認する
- 仕訳確定を経理担当者が確認する
- 会計連携用の CSV を出力する

## 操作の流れ

このシステムは、主に次の流れで動きます。

1. 初期設定を準備する
2. 証憑ファイルをアップロードする
3. OCR 結果を確認しながら伝票を作成する
4. 申請情報と仕訳明細を入力して申請する
5. 支払承認を進める
6. 経理承認を進める
7. 必要に応じて入出金 CSV や会計出力を行う

## 想定している利用者

- 一般利用者: 書類の登録、申請、差戻し対応を行う
- 承認者: 支払申請の確認、承認、却下、再申請確認を行う
- 経理担当者: 経理観点の確認、入出金処理、会計出力を行う
- 管理者: 組織、権限、マスタ、契約情報などを管理する
