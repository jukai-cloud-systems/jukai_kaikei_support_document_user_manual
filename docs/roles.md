# ロール別の使い方

## 一般利用者

主に使う画面:

- ホーム
- ファイルアップロード
- 基本情報入力
- 申請情報入力
- 申請承認一覧

主な作業:

1. 証憑をアップロードする
2. 下書きを作成する
3. 申請情報と明細を入力する
4. 申請を提出する
5. 却下時は修正して再申請する

## 承認者

主に使う画面:

- 申請承認一覧
- 承認状況一覧
- 基本情報確認
- 申請情報確認

主な作業:

1. 申請中の伝票を確認する
2. 内容に問題がなければ承認する
3. 不備があれば却下する
4. 必要に応じて再申請を確認する

## 経理担当者

主に使う画面:

- 経理承認一覧
- 銀行 CSV 取込
- 入出金取引編集
- 会計出力

主な作業:

1. 承認済み伝票を確認する
2. 会計観点で内容を補正する
3. 経理承認する
4. 入出金データと対応付ける
5. 会計出力を行う

## 管理者

主に使う画面:

- 取引先設定
- 書類種別設定
- 決済種別設定
- 会計処理方法設定
- 承認者設定
- 承認上限設定
- 設定
- 管理画面

主な作業:

1. 初期マスタを整備する
2. 変更依頼に応じてマスタを更新する
3. ユーザーや部門の変更に追従する
4. 契約会社や管理者情報を管理する
