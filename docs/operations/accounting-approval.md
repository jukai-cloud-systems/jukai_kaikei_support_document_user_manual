# 経理承認

## 経理承認で行うこと

経理担当者は、支払承認後の伝票を会計観点で確認します。

確認対象:

- 基本情報の妥当性
- 申請情報の妥当性
- 明細の税区分
- 補助コード
- 会計処理方法
- 出力対象として問題がないか

## 基本的な流れ

1. 経理承認一覧を開く
2. 対象伝票を選ぶ
3. 必要に応じて基本情報を修正する
4. 必要に応じて申請情報や明細を修正する
5. 経理承認する

## 経理担当者が見るべき観点

- 伝票の合計と明細合計が一致しているか
- 決済種別と税区分が正しいか
- 会計処理方法から補助コードが適切に引けるか
- 支払方法と入出金処理方法が矛盾していないか

## 入金確認・支払確認

画面上では、入金確認や支払確認に相当する処理を記録する構成があります。  
実運用では、銀行明細や社内ルールと照合して更新してください。

## 承認後に進む処理

経理承認後は、次の処理に進みます。

- 入出金 CSV との対応付け
- 会計出力
- 出力済み管理
