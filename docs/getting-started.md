# はじめに

## システムの基本構造

このシステムは、会社ごとの環境で利用する会計支援システムです。  
利用者は所属会社の環境にログインし、担当業務に応じて次の処理を行います。

- 支払伺い作成
- 支払承認／確認
- 口座振替FD作成
- 仕訳確定
- 仕訳データ出力

## 利用開始前に必要なもの

- ログイン用アカウント
- 所属部門
- 必要に応じた承認権限または経理権限
- 会社で利用する各種マスタ

## 代表的な利用権限

## 一般利用者

- ファイルをアップロードする
- 伝票を下書き保存する
- 申請を提出する
- 差戻し後に修正して再申請する

## 承認者

- 部門に届いた申請を確認する
- 承認または却下を行う
- 承認状況一覧を確認する

## 経理担当者

- 経理観点で申請内容を確認する
- 必要に応じて修正する
- 経理承認を行う
- 入出金 CSV を取り込む
- 会計連携用 CSV を出力する

## 画面の分類

- ホーム: 自分に関係する申請情報を確認する
- 支払伺い作成: 書類の取込みと支払申請を行う
- 支払承認／確認: 登録済みの伝票を検索・確認する
- 経理業務: 仕訳の確定や口座振替FDの作成を行う
- 各種マスタ: ユーザー、取引先、勘定科目、承認者設定などを管理する

## 注意事項

- 下書きのままでは承認フローに流れません
- 申請後に修正が入ると、履歴管理のため旧データが非表示扱いの履歴になる場合があります
- 会計出力や支払処理の前に、承認状態と入出金状態を確認してください
